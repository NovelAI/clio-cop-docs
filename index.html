clio-cop

An automated content-moderation bot that protects Discord communities from image-based scams.

What it does

clio-cop detects a recurring crypto-impersonation scam in which bad actors post a fixed sequence of images — a fake celebrity endorsement, a phishing-site screenshot, a fake "withdrawal success" screen, and a fake wallet balance — across channels to defraud members.

  • Passively scans image attachments on posted messages.
  • Scores them against known scam-image signatures using perceptual-hash and CLIP image-embedding similarity over the ordered image "story".
  • Removes high-confidence scam posts and bans the poster; flags borderline cases to human moderators.
  • Detects the same image set reposted across multiple channels in quick succession.

Privacy at a glance

  • Uses the Message Content intent solely to read image attachments — it does not read or store message text.
  • Images are processed in memory and discarded; no database, no long-term storage of user content.
  • The only retained record is moderation evidence posted to a private, moderators-only log channel inside your own server.
